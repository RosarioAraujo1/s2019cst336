A red black tree is a self balancing binary tree that requires every node to follow the rules to maintain the balance of the tree.
  • 1. Every Node in the tree must be red or black.
  • 2. The Root of the tree is always black.
  • 3. No two red nodes could be adjacent to each other.
  • 4. The path from a node to a null node must have the same black height.
